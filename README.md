# Калькулятор кондитера

Веб-приложение для расчёта себестоимости кондитерских изделий и рекомендуемой цены продажи. Подходит для домашних кондитеров и небольших производств.

## Возможности

- Расчёт стоимости ингредиентов по цене и весу упаковки
- Учёт упаковки, декора и прочих производственных расходов
- Коммунальные услуги, амортизация, транспорт, запас на потери
- Стоимость работы кондитера (часы × ставка)
- Наценка и округление цены продажи
- Себестоимость партии, одной штуки и 100 граммов
- Светлая и тёмная тема (сохраняется в браузере)
- Сохранение позиций упаковки и декора в `localStorage`

## Запуск

Приложение работает без сборки и сервера — откройте `index.html` в браузере.

**Windows (PowerShell):**

```powershell
Start-Process index.html
```

**Или через локальный сервер (опционально):**

```bash
npx serve .
```

После запуска откройте адрес, который покажет `serve` (обычно http://localhost:3000).

## Использование

1. Откройте главную страницу и нажмите «Открыть калькулятор».
2. Заполните основную информацию: название, тип, количество и вес.
3. Добавьте ингредиенты с ценой упаковки и использованным весом.
4. Укажите упаковку, декор и производственные расходы.
5. Введите время работы и наценку.
6. Нажмите «Рассчитать себестоимость».

После первого расчёта результат обновляется автоматически при изменении полей.
